## Sobre o projeto

Este é o meu portfólio pessoal, desenvolvido com foco em uma interface moderna, responsiva e profissional.

O objetivo do projeto é apresentar minha identidade profissional, minhas principais tecnologias, minha área de atuação e meus projetos em destaque, com uma experiência visual elegante, fluida e adaptada para diferentes dispositivos.

## Deploy

- JavaScript

## Funcionalidades

- Layout moderno e responsivo
- Tema claro e escuro
- Suporte a múltiplos idiomas
- Seção de apresentação profissional
- Seção sobre mim
- Lista de tecnologias/skills
- Projeto em destaque
- Área de contato
- Animações suaves com Framer Motion
- Deploy em produção pela Vercel
